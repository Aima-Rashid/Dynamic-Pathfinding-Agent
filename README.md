# Dynamic Pathfinding Agent

## Description
This project implements A* and Greedy Best-First Search (GBFS)
with support for Manhattan and Euclidean heuristics.
It also includes an optional dynamic obstacle mode and a visual grid interface using Tkinter.

## Features
- A* Search
- Greedy Best-First Search
- Manhattan & Euclidean Heuristics
- Dynamic Obstacle Mode
- Visual Grid Interface (Tkinter)

## Requirements
- Python 3.10 or higher
